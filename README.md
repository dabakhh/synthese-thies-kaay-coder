#  Projet d'Immersion Web - [Thies Kaay Ñu Coder]

Ce projet a été réalisé dans le cadre d'une immersion intensive d'une semaine Chez **Wommate Technology**. L'objectif était de construire le site vitrine de l'événement en travaillant en équipe via Git/GitHub.

##  Liens utiles
- **Démo en ligne (Netlify) :** [https://thies-kaay-nu-coder.netlify.app/] (Visiter le site en ligne)
- **Ma contribution principale :** Page Contact & Validation

##  Mes Réalisations
Au cours de ce projet, j'ai eu la responsabilité de plusieurs aspects clés du frontend :

### 1. Développement & Validation de Formulaire
- Création de la **page Contact** complète.
- Mise en place d'une **validation dynamique** en JavaScript (utilisation de Regex pour l'email, gestion des messages d'erreur en temps réel).

### 2. UI/UX & Design
- **Section Programmes :** Création et stylisation des cartes (cards) présentant les différentes formations.
- **Optimisation de l'Accueil :** Amélioration visuelle de la page principale.
- **Responsive Design :** Adaptation de l'interface pour une expérience fluide sur mobile et tablette.

### 3. Coordination Technique
- **Gestion de projet Git :** J'ai coordonné le rassemblement des travaux de différents groupes, géré les conflits de fusion (merge) et structuré le projet final.

##  Technologies utilisées
- **HTML5 / CSS3** (Flexbox, Grid)
- **JavaScript (ES6+)**
- **Git & GitHub** (Workflow par branches)
- **Netlify** (Déploiement continu)
